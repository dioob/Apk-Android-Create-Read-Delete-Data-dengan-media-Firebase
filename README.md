# Apk Android Create,Read & Delete Data dengan media Firebase

<p align="justify">Membuat aplikasi untuk menyimpan data, membaca, dan menghapusnya. Aplikasi ini menggunakan media penyimpanan Firebase Database, Firebase Storage, dengan autentikasi sebagai admin menggunakan Firebase Authentication.<p>
</br>
<p align="center">
  <img width="340" height="600" src="https://github.com/dioob/Apk-Android-Create-Read-Delete-Data-dengan-media-Firebase/blob/master/Screenshot_33.png">
</p>
</br>
<p align="justify">Aplikasi ini menggunakan 2 metode pengguna untuk masuk yaitu sebagai pengunjung dan sebagai admin. Bila sebagai pengunjung, pengguna hanya dapat melihat daftar dan info tanpa dapat melakukan input data maupun menghapus data.<p>
</br>
<p align="center">
  <img width="340" height="600" src="https://github.com/dioob/Apk-Android-Create-Read-Delete-Data-dengan-media-Firebase/blob/master/Screenshot_34.png">
</p>
</br>
<p align="center">
  <img width="340" height="600" src="https://github.com/dioob/Apk-Android-Create-Read-Delete-Data-dengan-media-Firebase/blob/master/Screenshot_35.png">
</p>
</br>
<p align="justify">Sedangkan untuk masuk sebagai admin, pengguna diharapkan untuk mengisi email & password. Setelah login akan diberikan fitur untuk menambah dan menghapus data. Menambah data dengan data foto akan ditampilkan pilihan mengambil gambar melalui foto langusng menggunakan kamera atau memilih dari galeri HP. Gambar inilah yang akan disimpan di media Firebase Storage dengan menyimpan linknya ke dalam Firebase Database pada masing-masing data.<p>
